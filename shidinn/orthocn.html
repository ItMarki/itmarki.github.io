希頂字母是以字母本身發音加上其“漢寫體”作為一個漢字在“希頂漢語”中的發音，兩者拼合起來命名的。例如：希頂字母“A”的“漢寫體”作“大”，在“希頂漢語”中“大”字的讀音用希頂字母記作“da”，因此“A”的名稱為“A-da”——字母名稱的中間需帶上連字符“-”以區隔兩個部分，否則，如寫作“Ada”則變成了一個字（詞）而不是字母了。

希頂字母的形狀與美國原住民切羅基部族文字學家塞闊雅在19世紀發明的切羅基文字相近，然而，希頂字母和切羅基文在深層次並不相似：切羅基文是音節文字，而希頂字母除了單個字母組成音節之外，還效仿了漢語拼音及一些元音附標文字的做法，以多個字母相拼形成音節。

“希頂字母”這個詞語，可以用希頂字母寫作 ⇧xdi8 ⇧aho ⇧fHLmA（上圖第一行藍色文字），也可用希頂漢字寫作“灯⿰纟火字母”（第二行黑色文字），也可用希頂形音漢字寫作“⿰火⿳刀工月⿰纟⿱九口⿱⿱宀下尺⿱巾女”（第三行橘色文字）。

希頂漢字的構思與已廢除的越南喃字（“字喃”）相近。越南喃字從漢語中“轉移”而創造的新字主要是遵循“形聲”方式，如“母親”在越南語中為“美”聲（mẹ），又在漢字中為“女”部，故喃字寫成“{女美}”。希頂漢字採用類似喃字的規律，但不同的是，希頂漢字中的部首也是發音的——就如假設漢語有這個喃字對應的{女美}，卻不讀 měi，而讀 nǚ měi（希頂構擬： AmeV，以A是女性相關部首故）。

由上可知，一個希頂漢字（或一個希頂字母語言音節）是由“部首”和“聲部”（“希頂華語”中為“形部”）組成。如：“思念”的“思”字，“希頂漢語”中作 siF（si 從現代漢語 sī 而來，再加上一個代表內心情感的部首 F），而“希頂華語”中則把整個字孤立出來作為一個“形部”，“思”讀作 nua。在“希頂華語”中，希頂漢字的字形與現代漢語讀音已經完全沒有關係了，其讀音完全遵循黃雀飛自造的一個（據稱）有1400多個字的對照表來發音；依此，我們也可以合成新的希頂語單字，例如“偲”： Hnua（表示人物相關的部首“人” + 形部“思”nua）。同時，在形部的對照表中還存在一些“假借”的現象，如“朋友”的“友”假借“發展”的“發”（簡體：发）為同一個形部，這使漢語拼音和希頂形部之間更難找到對應的規律——但也不失為希頂語的特色了。

關於希頂漢字與希頂字母書寫系統之間的轉換，有兩種形式：一是“形聲”，一是所謂的“象形”。 “形聲”就如之前舉的例子那樣，取“部首字母”和“聲部”（或“形部”）相拼，有如：
房 ＝ 户——建築或穿著 (p) + 方 ( fT) ＝ pfT
玲 ＝ 王——玉石相關 ( 3) + 令 ( li8) ＝ 3li8
晨 ＝ 日——自然現象 (b) + 辰 ( Bi8) ＝ bBi8
（注意：希頂語中“部首”其實是指作為部首使用的希頂字母，與漢字部首的形態並無關聯。如：“自然現象”的部首是“b”，無論漢字寫成“日”還是“白、風、光”等都是同一個部首。）

“形聲”構字法中，部首在前表示這個部首直接與“聲部”（或“形部”）相拼；部首在後則其“聲部”（“形部”）代表整體發音。例如：
瓦 ＝ wa
咓 ＝ owa (口 + 瓦)
哇 ＝ wao (“哇” + 口)
這種“編碼”方式讓希頂語具有了一定的“漢字輸入法”的意味。當然，這種方式在帶來靈活性的同時，也給希頂語內在解讀帶來了雜糅現象。就如上面看到那樣，生僻字“咓”和常用字“瓦、哇”竟然“同居一室”——不過不失為希頂語的特點。

此外雜糅現象更嚴重的是簡繁體問題，因為同一個漢字的簡體、繁體和異體，只要寫法不同，在希頂字母中就有“唯一的編碼”。例如：
爱 ( 4YF) ←→ 愛 ( 4YFH)
杀 (sa) ←→ 殺 ( 7sa)
對“不規則的象形字和特殊象聲詞”（黃雀飛語），希頂語還有所謂“象形（以及“象聲”）的“編碼方式”。黃雀飛自身舉了兩個例子，有如：
香 ＝ h（禾部）+ T（漢寫體為“日”，形狀與“日”相似）＝ hT。此為“象形”。
乒乓 ＝ 拼音 pīngpāng ＝ pi8pT → 加“擬聲詞部首”（口部，o）→ ⇧opi8⇧opT。據黃雀飛自述，此“O”為中寫體——希頂字母除了大小寫體和漢寫體外尚有“中寫體”，但很少出現實例。原文作者 Raymond 認為如果不能使用中寫體，則用連字符“-”接詞寫作 o-pi8pT也可以。

最後需注意，希頂字母語言書寫採用以語素為單位的分詞方式（即：“以 語素 為 單位 的 分詞 方式”）。因此有時會讀錯位，如“ 53Fyu2”（心愿）由“ 53”（心）和“ Fyu2”（愿）組成。如區隔為“ 53F”和“ yu2”則出現了不存在的字（詞），這種現象黃雀飛稱之為“假字（詞）”。原文作者 Raymond 認為避免出現“假字（詞）”的方法是在必要處用連字符“-”區隔，作 53-Fyu2”。書寫者可視情況使用連字符號避免出現這種現象。

希頂漢字的自造字

希頂漢字”是希頂語的漢字書寫系統，裡面有許多黃雀飛的自造字（具體個數不詳）。之前所舉的“{纟火}”字就是其中一個。這可以類比日本的國字——但是希頂語“造字”相對較靈活，原則上，只要是由部首和漢字部件組成的字，都可以用希頂字母“擬拼”出來，創造出任意的“新字”，如下

扻：8q2 (部首“扌”→ 8；“欠”音→ q2)
⿺辶歹：zdY (部首“辶”→ z；“歹”音→ dY)
⿰亲又：q82H (“亲”音→ q8；象形“又”→ 2；填補無部首字的還原符： H)
⿱⿱⺍冖卬：3fTH (象形“⺍”→“三”→ 3；部首“冖”→ f；“卬”音→ T；填補無部首字的還原符：H)

就像五筆輸入法那樣，希頂字母基本上可以在四碼之內把新出現的漢字都“編碼”完。基於類似的想法，希頂字母的創作者黃雀飛還提出了“用希頂字母代替英語作為計算機編程的基底字母”的構想，這成為了“希頂學”的一個重要部分。值得一提的是，這些新的漢字中大部分在希頂語中還未有含義，如果黃雀飛在其後的發展中賦予了每個類似漢字意義，那麼希頂語的表意興許會大大擴張。
